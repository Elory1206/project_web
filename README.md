# 프로젝트명
    휴대폰 비교 

<pre>
    프로젝트 설명(최소 5줄)
        1. 출고가 비교및 중고시세 비교
        2. 휴대폰 사양확인
        3. 브랜드별 잘나가는 제품 순위
        4. 가성비 좋은 제품 비교및 순위 제공
        5. 고객센터
            5-1 자주하는 질문 
            5-2 공지사항
            5-3 구매후기


</pre>

<pre>
    프로젝트 목적 (최소 5줄)
        고객이 항상 휴대폰의 출고가와 중고 시세를 확인할수 있다.
        또한 브랜드별 잘나가는 제품순위를 제공하고
        개인적으로 휴대폰을 추천 받을 수 있는 서비스를 제공할
        뿐만아니라 고객들에 자주하는 질문이나 약정들을 확인할 수 있다.
        궁극적으로 브랜드들을 많이 노출시켜 구매를 유도하는것이 목표이다.        
</pre>
